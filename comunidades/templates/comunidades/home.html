Cómo usar la API:

Método GET

  • .../comunidades/ : permite ver una lista de todas las comunidades
  • .../comunidad/1 : (1 es el número de id de la comunidad) permite ver una comunidad especificando su id
  • .../delegadas/ : permite ver una lista de todas las delegadas
  • .../delegada/1 : (1 es el número de id de la delegada) permite ver una delegada especificando su id
  • .../productos/ : permite ver una lista de todos los productos
  • .../producto/1 : (1 es el número de id del producto) permite ver un producto especificando su id
  • .../pedidos/ : permite ver una lista de todos los pedidos
  • .../pedido/1 : (1 es el número de id del pedido) permite ver un pedido especificando su id
